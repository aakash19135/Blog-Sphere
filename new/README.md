# Medium Clone Frontend

A beginner-friendly Medium-inspired blog website built using HTML, CSS, and JavaScript. This project focuses on creating a clean and responsive blogging platform interface with features commonly found in modern blog websites.

## Features

- Responsive Navigation Bar
- Medium-inspired User Interface
- Blog Cards with Author Information
- Read Time Display
- Like (Clap) Counter UI
- Comment Count UI
- Bookmark Button UI
- Trending Topics Sidebar
- Responsive Design for Mobile and Desktop
- Infinite Scroll Support (Optional)
- Frontend Only (No Backend)

## Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)
